Burp suite is an integrated set of tools for pentesting web applications.

## Components
* **Comparer**: Compares different responses.
* **Decoder**: Encodes/Decodes/Hashes data.
* **Extender**: Adds additional functionality to Burp Suite.
* **Intruder**: Automated attacks against target.
* **Proxy**: Proxy for channeling browser traffic through Burp Suite.
* **Repeater**: Sends multiple repeated requests to target.
* **Scanner**: *Pro Version Only* Automated tests.
* **Sequencer**: Tests randomness.
* **Target**: Scoping and site-map detection.
