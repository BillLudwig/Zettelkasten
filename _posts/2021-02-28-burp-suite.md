Burp suite is an integrated set of tools for pentesting web applications.

## Components
* [Comparer]({% post_url 2021-02-28-burp-suite-comparer %}): Compares different responses.
* [Decoder]({% post_url 2021-02-28-burp-suite-decoder %}): Encodes/Decodes/Hashes data.
* [Extender]({% post_url 2021-02-28-burp-suite-extender %}): Adds additional functionality to Burp Suite.
* [Intruder]({% post_url 2021-02-28-burp-suite-intruder %}): Automated attacks against target.
* [Proxy]({% post_url 2021-02-28-burp-suite-proxy %}): Proxy for channeling browser traffic through Burp Suite.
* [Repeater]({% post_url 2021-02-28-burp-suite-repeater %}): Sends multiple repeated requests to target.
* Scanner: *Pro Version Only* Automated tests.
* [Sequencer]({% post_url 2021-02-28-burp-suite-sequencer %}): Tests randomness.
* [Target]({% post_url 2021-02-28-burp-suite-target %}): Scoping and site-map detection.
